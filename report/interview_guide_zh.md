# 项目二：太阳能板角度优化模拟 —— 面试要点速记（给 Jayden 准备）

如果面试官问起这个项目，你只需要用讲故事的口吻，把下面的逻辑顺出来就行，核心是展现你**用物理和数学知识解决实际问题**的能力。

---

## 1. 动机：你为什么会做这个项目？
**核心逻辑：** 这是上一个环保项目的“下集”，从发现问题到解决问题。
> "在我做完大湾区空气质量分析之后，我深刻意识到化石燃料污染是很严重的问题。刚好我在 A-Level 学了物理（能量、光）和数学（三角函数），我就在想：**我能不能用自己学的知识，为深圳的清洁能源找找办法？** 太阳能是最常见的清洁能源，但我很好奇，在深圳这个特定的纬度（22.5° N），太阳能板到底倾斜多少度发电效率最高？为了解答这个问题，我做了这个模拟项目。"

## 2. 数据与方法：你是怎么模拟的？
**核心逻辑：** 强调这是“纯数学和物理推导”，不是网上随便抄的代码。
> "这个项目没有用现成的数据集，而是**完全基于物理公式和天文模型**用 Python 写出来的。
> 首先，我用了天文学公式算出了一年中每一天太阳的赤纬角（Declination angle）和高度角（Elevation angle）。
> 然后，我用三角函数（Trigonometry）计算了阳光直射在不同倾斜角度的面板上，到底能吸收多少能量（跟夹角的余弦值 cosine 有关）。我用代码跑了一个循环，模拟了全年 365 天、从 0 度（平放）到 90 度（垂直）的所有情况。"

## 3. 你的核心发现是什么？（最重要！）
**核心逻辑：** 给出明确的数字结论，并对比不同的策略。
> "我的模型得出了两个非常有意思的结论：
> 
> **发现一（最优固定角度）：** 在深圳，如果你想把太阳能板固定死，全年发电量最大的角度是 **19度（朝南）**。这刚好跟深圳的纬度（22.5度）差不多，符合工程学的经验法则，这证明我的数学模型是准确的。
>
> **发现二（经济学/策略反思）：** 我比较了三种策略：1是永远固定在19度；2是买昂贵的自动追踪支架每天跟着太阳转；3是纯手工按照春夏秋冬调4次角度。
> 结果发现，手工调4次比永远固定多发 **5.4%** 的电；而昂贵的自动追踪支架只比手工调多发了 **1.4%** 的电！所以从经济效益来看，对普通家庭来说，**不用买昂贵的自动支架，只需要每换季时手动调一下角度，性价比是最高的！**"

## 4. 局限性与反思（Critical Thinking）
**核心逻辑：** 承认模型是理论上的，现实更复杂。
> "当然，我知道我的模型只是一个**“理论上的晴天模型”（Clear-sky model）**。它没有考虑到深圳夏天的台风和梅雨季节的云层遮挡（漫反射辐射）。如果未来要在大学里深入研究，我会把我第一个项目里收集到的真实天气数据（比如下雨天数）结合到这个物理模型里，做一个更贴近现实的复合模型。"

---
**💡 终极建议：**
这个项目最大的亮点是你的**逻辑闭环**（计算最优角度 -> 对比不同策略的经济效益 -> 发现自动追踪设备性价比不高）。向面试官强调这个结论，会让你显得既有扎实的理科基础，又有非常敏锐的现实商业思维。
